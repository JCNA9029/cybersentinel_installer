# cybersentinel_installer

**Recommended Specs (For AI Analyst)** 

GPU: 8GB VRAM (e.g., NVIDIA RTX 3050 8GB or RTX 4060 8GB)

System RAM: 16GB DDR4

Storage: NVMe M.2 SSD (Mandatory. Loading an LLM from a standard HDD or SATA SSD causes heavy latency and timeout errors).

(This worked on my laptop: 1050ti, i5 8th Gen but very slow/bottlenecked)

**Instructions**

Install [Inno Setup](https://jrsoftware.org/isdl.php).

Run CyberSentinel_setup.iss 

Install

Go to Output -> run the .exe

Finished
